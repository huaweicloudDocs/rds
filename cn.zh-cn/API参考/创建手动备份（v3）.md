    Microsoft SQL Server实例：

    ```
    {
    "instance_id": "d8e6ca5a624745bcb546a227aa3ae1cfin04",
    "name": "backup",
    "description": "mannual backup",
    "databases": [
    {
    "name": "db1"
    },
    {
    "name": "db2"
    }
    ]
    }
    ```

    MySQL/PostgreSQL实例：

    ```
    {
    "instance_id": "d8e6ca5a624745bcb546a227aa3ae1cfin01",
    "name": "backup",
    "description": "mannual backup"
    }
    ```
