-   响应样例

    自动备份策略关闭时：

    ```
    {
    	"backup_policy": {
    		"keep_days": 0
    	}
    }
    ```

    自动备份策略开启时：

    ```
    {
    	"backup_policy": {
    		"keep_days": 7,
    		"start_time": "19:00-20:00",
    		"period": "1,2"
    	}
    }
    ```
