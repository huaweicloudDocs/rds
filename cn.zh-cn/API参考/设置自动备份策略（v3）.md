-   请求样例

    更新自动备份策略：

    ```
    {
    "backup_policy": {
    "keep_days": 7,
    "start_time": "19:00-20:00",
    "period": "1,2"
    }
    }
    ```

    关闭自动备份策略：

    ```
    {
    "backup_policy": {
    "keep_days": 0
    }
    }
    ```
