-   接口约束
    -   目前支持如下数据库引擎：
        -   MySQL
        -   PostgreSQL
        -   Microsoft SQL Server

    -   修改的参数必须在指定数据库版本的默认参数范围内，具体可配置范围请参见《关系型数据库用户指南》中“编辑参数组”的内容。
    -   修改参数组会应用到所有已关联的实例，并根据修改后的参数是否需要立即生效重启实例。
