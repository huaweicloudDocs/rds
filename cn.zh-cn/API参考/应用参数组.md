-   接口约束

    目前支持如下数据库引擎：

    -   MySQL：仅支持5.6，5.7。
    -   PostgreSQL：仅支持9.5，9.6。
    -   Microsoft SQL Server：仅支持2014 SE、2014 EE、2016 SE、2016 EE、2008 R2 EE、2008 R2 WEB、2014 WEB、2016 WEB。
