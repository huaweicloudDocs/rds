## 功能介绍<a name="section4850156117316"></a>

-   URI格式

    PATH：/rds/v1/\{project\_id\}/datastores/versions/\{datastore\_version\_id\}/parameters/\{parameter\_name\}

    Method：GET

    ```
    }
    ```
