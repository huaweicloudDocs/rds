# 设置配置参数<a name="zh-cn_topic_0034973638"></a>

## 功能介绍<a name="section4850156117316"></a>

设置实例参数。

>![](public_sys-resources/icon-note.gif) **说明：**   
>设置实例参数时，系统将根据需要创建一个与该实例同名的参数组。
