# 查询数据库<a name="rds_06_0008"></a>

## 功能介绍<a name="section4850156117316"></a>

## URI<a name="section28961517113719"></a>

-   URI格式

    PATH：/\{version\_id\}/\{project\_id\}/instances/\{instance\_id\}/database

    Method：GET
