# 获取备份下载链接<a name="rds_09_0006"></a>

## 功能介绍<a name="section4850156117316"></a>

获取备份列表。

## URI<a name="section28961517113719"></a>

-   URI格式

    PATH：/\{version\_id\}/\{project\_id\}/backup-files?backup\_id=\{backup\_id\}

    Method：GET

-   接口约束

    该接口支持下载三款引擎\(MySQL, PostgreSQL, Microsoft SQL\)的全量备份文件，以及MySQL的增量备份文件。


## 样例<a name="section43121332947"></a>

GET /v3/97b026aa9cc4417888c14c84a1ad9860/backup-files?backup\_id=c0c9f155c7b7423a9d30f0175998b63bbr01
