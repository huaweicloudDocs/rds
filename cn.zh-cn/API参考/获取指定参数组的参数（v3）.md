# 获取指定参数组的参数（v3）<a name="rds_09_0307"></a>

## 功能介绍<a name="section5782830193116"></a>

获取指定参数组的详细信息。

## URI<a name="section27821830203110"></a>

-   URI格式

    PATH：/\{version\_id\}/\{project\_id\}/configurations/\{config\_id\}

    Method：GET
