# 查询数据库引擎的版本<a name="rds_06_0001"></a>

## 功能介绍<a name="section61759636"></a>

获取指定数据库引擎对应的数据库版本信息。

## URI<a name="section18965813"></a>

-   URI格式

    PATH：/\{version\_id\}/\{project\_id\}/datastores/\{database\_name\}

    Method：GET
