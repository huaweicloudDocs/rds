# 获取所有实例规格信息<a name="zh-cn_topic_0032347783"></a>

## 功能介绍<a name="section25483595"></a>

获取指定数据库ID和区域下的所有实例规格信息。

## URI<a name="section28025763"></a>

-   URI格式

    PATH：/rds/\{versionId\}/\{project\_id\}/flavors

    Method：GET
