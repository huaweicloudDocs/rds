# 调整实例容量<a name="rds_06_0617"></a>

## 功能介绍<a name="section4850156117316"></a>

## URI<a name="section28961517113719"></a>

-   URI格式

    Method：POST

-   接口约束
    1.  调整实例容量高于原来容量，扩容容量要为10的倍数。
    2.  备实例大小和主实例大小一致，当主实例扩容时，会同时扩容备实例。
    3.  实例状态为服务中时可以进行扩容。
