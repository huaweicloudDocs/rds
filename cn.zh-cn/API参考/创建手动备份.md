-   接口约束

    该接口中databases参数是可选参数，只支持实例的数据库引擎Microsoft SQL Server。


-   请求样例

    ```
    {
    "backup": {
    "description": "My Backup",
    "instance": "44b277eb-39be-4921-be31-3d61b43651d7",
    "name": "backup"
    }
    }
    
    Microsoft SQL Server 创建局部备份 请求样例
    { 
    "backup": { 
    "description": "My Backup", 
    "instance": "c2aac04b4b44460aa75674fc0fd878b2no04", 
    "name": "test",
    "databases": [
            {
                "name": "user01"
            },
            {
                "name": "user02"
            }
        ]
    } 
    ```
