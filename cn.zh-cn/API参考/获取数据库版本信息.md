# 获取数据库版本信息<a name="zh-cn_topic_0032347782"></a>

## 功能介绍<a name="section61759636"></a>

获取指定类型的数据库版本相关信息。

## URI<a name="section18965813"></a>

-   URI格式

    PATH：/rds/\{versionId\}/\{project\_id\}/datastores/\{datastore\_name\}/versions

    Method：GET
