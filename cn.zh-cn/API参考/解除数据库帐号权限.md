# 解除数据库帐号权限<a name="rds_06_0014"></a>

## 功能介绍<a name="section4850156117316"></a>

在指定实例的数据库中，解除数据库帐号的权限。

## URI<a name="section28961517113719"></a>

-   URI格式

    PATH：/\{version\_id\}/\{project\_id\}/instances/\{instance\_id\}/db\_privilege

    Method：DELETE
